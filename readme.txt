
In the empirical_study folder, we provid the dataset of 5 Gerrit projects, one dataset containing all GitHub projects, and R scripts for regression model.
	RQ2_github.r is the R code for regression modeling for GitHub projects, and RQ2_gerrit.r is the R code for regression modeling for Gerrit project.
	
In the SLR folder, Search_string.docx presents the search string for SLR on IEEE, Google Scholar, ACM and Scopus search.
	List_of_papers.xlsx is the list of papers that we obtained in the subsequent steps of our SLR.
	inclusion_exclusion_criteria.docx presents the inclusion and exclusion criteria for our SLR.
	SLR_steps.docx presents the detailed steps of performing SLR.
